# 🐧 Unix & Linux Kernel Engineering (2008–2012)

### **Strategic Overview**
This repository archives the deep-level systems engineering performed during my tenure at **IBM India**. It focuses on mission-critical Unix and Linux-based environments (AIX and HP-UX) where I managed kernel-level upgrades, network resilience, and advanced volume management.

---

### **🏗️ Core Architectural Pillars**

* **AIX Lifecycle & Kernel Upgrades:** Orchestrated the strategic version upgrade of **AIX servers from 6.1 to 7.1**, managing binary compatibility, system consistency, and kernel-level performance tuning.
* **Network Resilience (Bonding):** Engineered and implemented **EtherChannel/Bonding** on Unix nodes to provide link aggregation and sub-second network fault tolerance.
* **Storage Virtualization:** Expert implementation of **Veritas Volume Manager (VxVM)** for complex disk management, mirroring, and high-performance storage virtualization.
* **Enterprise Unix Administration:** Managed mission-critical **HP-UX 11i v3** environments, ensuring 99.99% uptime for core enterprise workloads.

---

### **🚀 The "Evolution" Bridge**
Upgrading AIX kernels and implementing Veritas volume management provides the technical foundation required for modern **OpenShift** administration. Understanding how to manage bonding, kernel-level upgrades, and storage virtualization in Unix is the master-level precursor to managing secure, container-native infrastructure today.
